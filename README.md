# 🎓 EduTrack: Student Performance Analysis

A modern, interactive, and AI-powered dashboard designed to track, analyze, and optimize student academic performance. Built using **Flask**, **SQLAlchemy**, and integrated with **Google Gemini AI** for intelligent performance feedback.

---

## ✨ Key Features

- 📊 **Dynamic Analytics Dashboard**: Interactive visual reports on student metrics, grades, and progression.
- 🤖 **Gemini AI Integration**: Automated, AI-driven feedback and tailored study plans for students.
- 📝 **Data Entry & Management**: Easily add, edit, and maintain student records and exam scores.
- 📑 **Report Generation**: Automatically generate comprehensive presentation slides using the built-in PowerPoint generation tool.

---

## 🛠️ Technology Stack

- **Backend**: Python, Flask
- **Database**: SQLite (SQLAlchemy ORM)
- **AI Service**: Google Gemini API
- **Frontend**: HTML5, Vanilla CSS3, JavaScript
- **Presentation Export**: `python-pptx`
